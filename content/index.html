概要

HTML5自体はAPIやJSを使う、拡張的な土台としての進化したHTMLであり、CSS3はより拡張性を持ったサイトデザインができるCSSです。
極力画像を使わない、flashを使わない、JSを使わないで、リッチなコンテンツを作るために、HTML5とCSS3を使用したテクニックを紹介していきたいと思います。まずはCSS3の装飾から。ChromeとFirefoxで見え方は違います。webkit/moz指定も忘れずに。 対応ブラウザはChrome 10,Firefox 3.6,Safari 5.0,IE 9 以上です。

Gradation グラデーション

linear-gradientでは、線形のグラデーションが作成できます。

/* Firefox */
background: -moz-linear-gradient(top, #FFFFFF,  #91D0D8 3%, #003366);
/* Safari,Google Chrome */
background: -webkit-gradient(linear, left top, left bottom, from(#FFFFFF), color-stop(0.03, #91D0D8), to(#144976));
}

radial-gradientでは、放射状線のグラデーションが作成できます。

/* Firefox */
background: -moz-radial-gradient(top right, circle, #91D0D8 0px, #144976 35px, #32698D 40px, #144976 70px, #32698D 75px, #144976 110px);
/* Safari,Google Chrome */
background: -webkit-gradient(radial, right top, 0, right top, 110, from(#91D0D8), color-stop(35%, #144976), color-stop(40%, #32698D), color-stop(70%, #144976), color-stop(75%, #32698D), to(#144976));
